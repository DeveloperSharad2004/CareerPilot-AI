# 🚀 CareerPilot AI

An AI-powered Interview Preparation Platform built with the MERN Stack and Google Gemini AI.

The application analyzes a candidate's resume (or self-description) along with a job description to generate a personalized interview strategy, match score, skill gap analysis, technical questions, HR questions, and improvement suggestions.

---

## ✨ Features

- 🤖 AI-Powered Interview Report Generation
- 📄 Resume Upload (PDF/DOCX)
- 📝 Self Description Support
- 🎯 Job Description Analysis
- 📊 Resume Match Score
- 💡 Skill Gap Analysis
- ❓ Technical Interview Questions
- 👨‍💼 HR Interview Questions
- 📌 Personalized Interview Strategy
- 🔐 JWT Authentication
- 🍪 Secure Cookie-Based Login
- 📂 Interview History
- 📄 Downloadable Interview Report

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/DeveloperSharad2004/interviewiq-ai.git
```

### Install Dependencies

Backend

```bash
cd Backend
npm install
```

Frontend

```bash
cd Frontend
npm install
```

---

## Environment Variables

Create a `.env` file inside the Backend folder.

```env
PORT=3000

MONGODB_URI=your_mongodb_connection

JWT_SECRET=your_secret

GOOGLE_GENAI_API_KEY=your_api_key
```

---

## Run Project

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
